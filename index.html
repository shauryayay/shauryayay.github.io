Things I've built to solve real problems

CloudPilot

Self-service database provisioning API. One endpoint provisions production-ready Postgres, MySQL, MongoDB, or Redis clusters with GitOps audit trails and deterministic credentials.

  • Async provisioning pipeline: API returns a tracking ID, Celery worker drives the full lifecycle in the background
  • GitOps source of truth: every provision generates Helm config and commits it to Git for auditability
  • Deterministic secrets via SHA-256 derivation, so no passwords are ever stored in Git or config
  • Environment-aware defaults: dev gets starter instances, prod enforces deletion protection and multi-replica
  • Unified multi-engine API covering Postgres, MySQL, MongoDB, and Redis with the same endpoints

Deployer

ML model serving platform. Upload a trained sklearn, PyTorch, or TensorFlow model and it auto-generates a containerized FastAPI inference endpoint with health checks and async predictions.

  • Auto-framework detection: inspects file extensions and magic bytes to identify sklearn, PyTorch, or TensorFlow
  • Per-model containerization: each model runs in its own isolated Docker container with a dedicated FastAPI server
  • Async inference queue via Redis: predictions are queued and processed in the background, poll by job ID
  • Every model container exposes /health, /ready, and /metrics endpoints out of the box
  • Jinja2-templated Dockerfiles and serving code generated per framework
