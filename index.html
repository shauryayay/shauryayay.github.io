CloudPilot

Self-service database provisioning API. One endpoint provisions production-ready Postgres, MySQL, MongoDB, or Redis clusters with GitOps audit trails.

FastAPI Redis GitOps Docker
  • Async provisioning pipeline: API returns a tracking ID, background worker drives the full lifecycle
  • GitOps source of truth: every provision generates Helm config and commits it to Git for auditability
  • Deterministic secrets via SHA-256 derivation, no passwords stored in Git
  • Environment-aware defaults: dev gets starter instances, prod enforces deletion protection
  • Unified multi-engine API covering Postgres, MySQL, MongoDB, and Redis

Deployer

ML model serving platform. Upload a trained sklearn, PyTorch, or TensorFlow model and get a containerized FastAPI inference endpoint.

FastAPI Docker SDK Redis Jinja2
  • Auto-framework detection via file extensions and magic bytes
  • Per-model containerization: each model in its own Docker container with dedicated FastAPI server
  • Async inference queue via Redis: predictions queued and processed in background
  • Every container exposes /health, /ready, and /metrics endpoints
  • Jinja2-templated Dockerfiles and serving code generated per framework

Home About

Build. Break. Build it better.

I'm an engineer, and at heart, a problem-solver.

Give me something tangled, broken, or half-finished and I'm happy. The core of my work is getting to the bottom of a hard problem and building my way out of it: experimenting, breaking things, and rebuilding them a little better each time. I'm drawn to the space where backend engineering meets machine learning, but the technical details matter less to me than the craft underneath, turning something complicated into something that just works. And the feeling I chase is always the same, that moment when all the scattered pieces finally click into place.

I didn't always know I'd end up here. I fell for computer science because of a teacher, someone who made every class so alive and genuinely fun that I couldn't wait to show up. She turned what could have been dry into something I wanted to chase, and I've been chasing it ever since.

At heart, I've always been a puzzle person. I love getting to the core of a problem, experimenting my way toward a solution, then tearing it down and building it again a little better, the same impulse that had me endlessly rebuilding Lego sets as a kid. There's a specific kind of joy in watching something you made come together and hold.

When I'm not building, you'll usually find me with a book, hunting down a new coffee place, on a long walk with no particular destination, traveling somewhere I haven't been, or just spending unhurried time with family and friends.

I studied Computer Science with a minor in Data Science & Analytics at Amity University, where I earned a merit-based scholarship every year. But I don't think learning stops at a degree. I'm happiest when I'm a little out of my depth and figuring it out.

Thanks for reading, and for spending a little of your time here.
